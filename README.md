# Getting Started with Create React App

This project was bootstrapped with Create React App and typescript.
Styled components and antd for the ui.
axios for fetching the data.

## Available Scripts

In the project directory, you can run:

### `yarn`

to download the dependencies

### `json-server --watch employees.json`

to run the server on port 3000 or whatever

### `yarn start`

to run the react project on port 3001 or whatever

## The server and react app should be in two different ports
